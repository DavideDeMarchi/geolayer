Welcome to geolayer documentation!
==================================

**geolayer** is a Python library for the display of raster and vector datasets on the JRC BDAP (Big Data Analytics Platform).

It can be used in the JEO-lab environment (JupyterLab instance) to create notebooks that can display geo-spatial contentsm, or in the creation of Voil√† dashboards by using the `vois library <https://https://code.europa.eu/jrc-bdap/vois>`_ documented here: `vois library documentation<https://vois.readthedocs.io/>`_
